#git命令脚本集合:</br>
##push.sh
>直接进行master分支所有文件的提交，个人开发，基本没有冲突</br>
>运行命令  sh push.sh + "你提交的说明相当于commit -m"  或  ./push.sh + "你提交的说明相当于commit -m"
##mergePush.sh
ssss
